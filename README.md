# SCORA ER-14 ROBOT
## Objective

Apply forward and inverse kinematics to *4-DOF SCORA ROBOT* and perform pick and place movements.
## Components Used
ESP-32

DC Motors (24 V)

BTS7960 Motor Drivers

Incremental Encoders HEDS-5500#I14

End Effector-Servo Gripper
